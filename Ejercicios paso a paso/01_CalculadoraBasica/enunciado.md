# Ejercicio 1: Calculadora Básica

## Contexto
Imagina que trabajas en una tienda de electrónica y necesitas crear una calculadora simple para ayudar a los clientes a calcular precios con descuentos, sumas de productos, etc.

## Enunciado
Crea un programa en Java que funcione como una calculadora básica. El programa debe:

1. Mostrar un menú con las opciones: suma, resta, multiplicación y división.
2. Solicitar al usuario dos números.
3. Realizar la operación seleccionada.
4. Mostrar el resultado.
5. Manejar el caso especial de división por cero.

## Requisitos técnicos
- Uso de variables y tipos de datos primitivos (`int`, `double`).
- Uso de entrada/salida por consola con `Scanner`.
- Estructuras de control: `if-else` o `switch-case`.
- Operadores aritméticos básicos.

## Ejemplo de ejecución
```
=== CALCULADORA BÁSICA ===
Seleccione una operación:
1. Suma
2. Resta
3. Multiplicación
4. División
Opción: 1

Ingrese el primer número: 15.5
Ingrese el segundo número: 4.5
Resultado: 15.5 + 4.5 = 20.0
```

## Conceptos que aprenderás
- Declaración y uso de variables
- Tipos de datos primitivos
- Entrada de datos con Scanner
- Estructuras condicionales
- Operadores aritméticos
